# instalação

Instalação no Windows Powershell 🪟:

```powershell
winget install --id Docker.DockerDesktop
```

instalação via apt🐧:

```bash
sudo apt install docker-ce docker-ce-cli containerd.io
```

instalação no Mac:

```bash
brew install --cask docker
```
